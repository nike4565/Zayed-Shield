## 📝 **Installation: One Command**

```bash
# Clone HEARTSHIELD into your project
curl https://raw.githubusercontent.com/heartshield/core/main/install.sh | bash

# That's it. You're protected.
```

Or add to your `package.json`:

```json
{
  "dependencies": {
    "heartshield": "latest"
  },
  "scripts": {
    "shield:enable": "heartshield --mode=production",
    "shield:status": "heartshield --report"
  }
}
```

Then:

```bash
npm run shield:enable
# HEARTSHIELD activated. Your core is protected.
```
