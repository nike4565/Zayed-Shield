## 🤝 How to Contribute to the Roadmap

Anyone can influence what comes next.

### Submit a Feature Request

```
1. Open a GitHub Discussion in the "💡 Ideas" category
2. Use the title format: [ROADMAP] Your Feature Name
3. Explain: What · Why · Which phase it belongs to · How to implement
4. Community votes (👍) influence prioritization
5. Core Team reviews monthly and adds to roadmap if approved
```

### Pick Up an Open Task

Tasks marked `help wanted` or `good first issue` in Phase 3 are open for community contributors:

```bash
# Filter open roadmap tasks
https://github.com/nike4565/Zayed-Shield/labels/roadmap
https://github.com/nike4565/Zayed-Shield/labels/help%20wanted
```

### Sponsor a Feature

Organizations can sponsor specific Phase 4 or Phase 5 features for priority development. Contact: `nike494949@outlook.sa`
